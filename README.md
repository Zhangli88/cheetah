A lightweight event-driven library based on reactor pattern in development.<br>
<b>Features already supported</b>:<br>
  Regular file io events.<br>
  Socket io events.<br>
  Signal events.<br>
  Timer events.<br>
<b>Features in development</b>:<br>
  Asynchronous io events.<br>
<b>To test</b>:<br>
./configure<br>
make<br>
./benchmark
